Selamat datang di panduan CSS dasar. CSS adalah teknologi yang penting untuk mempercantik halaman web.

Apa Itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak halaman web.

Sintaks CSS

Sintaks dasar CSS terdiri dari selector dan deklarasi. Deklarasi berisi properti dan nilai yang ingin kita atur.

/* Sintaks dasar */
selector {
    properti: nilai;
}

Selector CSS

Beberapa selector yang umum digunakan dalam CSS adalah:

  • Selector Elemen: memilih elemen berdasarkan tag (contoh: `p`, `h1`).
  • Selector Kelas: memilih elemen berdasarkan kelas (contoh: `.kelas`).
  • Selector ID: memilih elemen berdasarkan ID (contoh: `#id`).

Contoh CSS

Contoh gaya CSS yang sederhana:

body {
    font-family: Arial, sans-serif;
    color: #333;
}

h1 {
    color: #007bff;
}

p {
    line-height: 1.6;
}
